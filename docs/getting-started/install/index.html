Installation

Wake is available for Windows, macOS, and Linux. Modules are available for Homebrew and most module managers popular on Linux systems

  • Windows
  • macOS
  • Linux
  • NixOS/Nixpkgs

Download and install the msi from our latest Release

On macOS, you can use Homebrew to install and update Wake.

brew install wake/tap/wake

You can install Wake using one of the module managers below.

RPM-based installation (RedHat/CentOS/Fedora)
dnf install https://releases.dl.wake.dev/wake_v0.0.1_amd64.rpm
DEB-based installation (Ubuntu/Debian)
curl -LO https://releases.dl.wake.dev/wake_v0.0.1_amd64.deb
sudo dpkg -i wake_v0.0.1_amd64.deb
APK-based installation (Alpine)
curl -LO https://releases.dl.wake.dev/wake_v0.0.1_amd64.apk
apk add --allow-untrusted wake_v0.0.1_amd64.apk

If you are using a distribution that does not use one of the module managers above, or require a 32-bit binary, check out additional download options attached to our latest release.

You can either use the module temporarily in a nix-shell:

nix-shell -p wake

Or install it globally by adding pkgs.wake to your environment.systemPackages.


Enjoy!
